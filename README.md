# 📇 Эмиль гений — английский по любимым YouTube-видео

**[→ Открыть приложение](https://cryptonerf.github.io/eng-project/)** · работает в браузере и как PWA на телефоне

Хотите смотреть любимые каналы на английском, но не хватает словарного запаса?
Вставьте ссылку на видео — приложение выгрузит субтитры и превратит их в колоду
карточек. Выучите слова — и вернитесь к видео подготовленным: вы уже знаете его
лексику, причём в тех самых фразах, где она звучит.

## Возможности

### 🃏 Карточки из любого видео

Ссылка на YouTube → готовая колода за полминуты. Каждая карточка — это слово,
перевод и **живые примеры из самого видео** с таймкодами. Повторные добавления
того же видео мгновенны: транскрипты кешируются для всех пользователей.

### ▶️ Фразы голосом спикера

Тап по таймкоду — и встроенный плеер проигрывает **ровно ту фразу из видео**,
где встретилось слово: старт с нужной секунды, автостоп в конце фразы, кнопка
«ещё раз». Слушайте живое произношение, не уходя из приложения. Плюс озвучка
любого слова и предложения синтезом (TTS).

### 🧠 Интервальные повторения (SRS)

Алгоритм SM-2, как в Anki: оценили карточку — она вернётся ровно тогда, когда
вы вот-вот её забудете. Кнопка **«Повторить сегодня»** на главной собирает
дневную порцию из всех видео сразу — каждая карточка приходит со своим
контекстом. Управление с клавиатуры: пробел — перевернуть, 1–4 — оценить.

### 📈 Прогресс, который видно

- **«Вы знаете X%»** — у каждого видео виден процент выученных слов и
  прогресс-бар: наглядно растёт готовность смотреть его без словаря.
- **Стрик** 🔥 — дни занятий подряд, счётчик выученного, повторения за сегодня.
- **Мой словарь** — все слова со всех видео: *изучаю / выучил / знаю*, с
  источниками и интервалами повторений.

### 🧹 Умный словарь

- **Лемматизация**: ran, running и runs — одна карточка «run» (неправильные
  глаголы и множественные числа включены). Подсветка в примерах находит все
  формы слова.
- **Фильтр мусора**: обрывки субтитров и не-слова в колоду не попадают.
- **Сложность по частотности**: цвет карточки = уровень слова (зелёный —
  простое, жёлтый — среднее, терракотовый — редкое). Простые и уже выученные
  слова скрываются, чтобы не мешать.
- **«Уже знаю»** — одно нажатие убирает слово из всех видео навсегда.

### 📱 Как приложение на телефоне

PWA: «Добавить на экран домой» — и это полноценное приложение со своей иконкой,
офлайн-режимом для повторений и синхронизацией. Вход по ссылке на почту или
через Google — прогресс общий на всех устройствах, аккаунт создаётся
автоматически при первом заходе.

### 📤 Экспорт

Колода выгружается в Anki (TSV с подсветкой примеров) или в CSV-таблицу.

## Как устроено

```text
Клиент (React + Vite + TS + Tailwind, PWA)  ←  GitHub Pages
   │
   ├── Firebase Auth        анонимный вход → привязка почты/Google
   ├── Cloud Firestore      колоды, глобальный прогресс слов, статистика,
   │                        офлайн-кеш из коробки
   └── Cloud Functions      ingest: yt-dlp достаёт субтитры, результат
                            кешируется в Firestore для всех пользователей
```

- Перевод EN→RU выполняется прямо в браузере (бесплатный эндпоинт Google
  Translate) и кешируется в данных пользователя.
- Прогресс слова — глобальный: выучили «immune» в одном видео — оно зачтено во
  всех остальных.
- Каждое видео скачивается с YouTube **один раз за всю жизнь приложения**
  (глобальный кеш транскриптов), дальше — мгновенно из базы.

## Разработка

```bash
npm run setup          # зависимости + бинарник yt-dlp (для legacy-сервера)
npm run dev            # клиент на localhost:5173 (бэкенд — облачный)
npm run build          # production-сборка клиента
npm run deploy:functions   # деплой Cloud Functions + правил Firestore
```

Деплой фронтенда автоматический: пуш в `main` → GitHub Actions → GitHub Pages.

## Структура

```text
eng-project/
├─ client/                  # React-приложение (PWA)
│  └─ src/
│     ├─ lib/               # words (карточки, лемматизация), srs, vocab,
│     │                     # repo (Firestore/localStorage), translate, anki
│     └─ components/        # WordCard, StudyView, ClipPlayer, Dictionary…
├─ functions/               # Cloud Functions: ingest (yt-dlp + PoToken + кеш)
├─ .github/workflows/       # автодеплой на GitHub Pages
└─ server/                  # legacy: локальный Express (не используется)
```
