# 📇 «Эмиль гений» — учи английский по YouTube

Вставьте ссылку на YouTube-видео — приложение выгрузит английские субтитры и соберёт
карточки со **словом**, **переводом** и **примерами в контексте** из видео.

> Репозиторий: eng-project. Дизайн: плоские «бумажные» карточки без градиентов,
> моноширинный шрифт (IBM Plex Mono), цвет карточки = сложность слова.

## Возможности

- 📥 **Субтитры с YouTube** — ручные или авто-сгенерированные (через `yt-dlp`).
- 🃏 **Карточки** слово / перевод / предложение-контекст с подсветкой слова.
- 🇷🇺 **Перевод** EN→RU через бесплатный Google Translate (без ключа).
- 🎯 **Фильтр по сложности** — частотный список прячет тривиальные слова.
- 🧠 **Интервальное повторение (SRS)** по алгоритму SM-2, прогресс хранится локально.
- 🔊 **Озвучка** слов и предложений (Web Speech API браузера).
- ⏱ **Таймкоды** — клик открывает нужный момент видео на YouTube.
- 📤 **Экспорт** в Anki (.txt/TSV) и таблицу (.csv).

## Стек

- **Клиент:** React + Vite + TypeScript + Tailwind CSS v4
- **Сервер:** Node + Express (извлечение субтитров через `yt-dlp`, прокси перевода)

## Запуск

```bash
# 1. Установить зависимости и скачать бинарник yt-dlp
npm run setup

# 2. Запустить сервер (:3001) и клиент (:5173) вместе
npm run dev
```

Откройте <http://localhost:5173>.

> `npm run setup` скачивает автономный бинарник `yt-dlp` в `server/bin/`
> под вашу ОС. Внешний Python не нужен.

## Структура

```text
eng-project/
├─ server/            # Express API
│  ├─ index.js        # маршруты /api/transcript, /api/translate
│  ├─ transcript.js   # извлечение и парсинг субтитров (yt-dlp + json3)
│  ├─ translate.js    # бесплатный EN→RU перевод с кешем
│  └─ bin/yt-dlp      # автономный бинарник (скачивается setup)
└─ client/            # React-приложение
   └─ src/
      ├─ lib/         # words, srs, storage, anki, tts, api, types
      └─ components/  # UI
```
